# Plan de Implementacion

Esta ruta existe como compatibilidad. En el modelo OpenSpec-first, los planes de implementacion de un cambio viven en `openspec/changes/<change-id>/design.md` y `tasks.md`.

## Uso Recomendado

Usar este archivo solo como enlace temporal al cambio activo. No mantener un plan paralelo.

```text
Plan vigente: `openspec/changes/<change-id>/design.md`
Tareas vigentes: `openspec/changes/<change-id>/tasks.md`
```

## Regla Breve

Un plan paralelo se desactualiza; el cambio OpenSpec debe concentrar el trabajo activo.
