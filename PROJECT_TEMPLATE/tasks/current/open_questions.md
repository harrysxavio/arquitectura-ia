# Preguntas Abiertas

Esta ruta existe como compatibilidad. En el modelo OpenSpec-first, las preguntas relevantes de un cambio deben registrarse en `openspec/changes/<change-id>/proposal.md`, `design.md` o `tasks.md`, segun corresponda.

## Uso Recomendado

Usar este archivo solo como enlace temporal si alguna herramienta espera la ruta. No mantener preguntas paralelas fuera del cambio activo.

```text
Preguntas del cambio: `openspec/changes/<change-id>/`
```

## Ejemplo Minimo

```markdown
Preguntas del cambio: `openspec/changes/add-request-priority/design.md`

No mantener preguntas paralelas aqui.
```

## Regla Breve

Una pregunta abierta debe vivir junto al cambio que puede resolverla.
