# Open Questions

> Plantilla. Se vuelve canonica solo dentro de un proyecto activo.

## Proposito

Registrar dudas que bloquean la ejecucion o cambian el alcance.

## Como llenarlo

Agregar solo preguntas que no se pueden resolver leyendo fuentes canonicas. Registrar respuesta y fecha cuando se resuelvan.

## Ejemplo minimo

```text
Pregunta: El rechazo de cliente bloqueado debe devolver 403 o 422?
Respuesta: 403, porque es una restriccion de permiso de negocio.
```
