# src/

Esta carpeta es ilustrativa y opcional.

No forma parte obligatoria del nucleo del framework. Existe para mostrar como un proyecto documentado puede convivir con un poco de codigo sin convertir `SAMPLE_PROJECT/` en una aplicacion completa.

Reglas de lectura:

- La verdad funcional vive en `docs/product/spec.md`.
- El diseno tecnico vive en `docs/architecture/sdd.md`.
- Las decisiones viven en `decisions/`.
- La memoria vigente vive en `memory/`.
- Este codigo solo ayuda a visualizar el flujo.
