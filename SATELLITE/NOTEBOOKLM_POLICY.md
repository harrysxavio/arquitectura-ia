# Politica de NotebookLM

NotebookLM sirve para investigacion externa y fuentes largas. No es una fuente oficial del proyecto.

Si la investigacion se vuelve durable, resumirla en OpenSpec, arquitectura, decisiones o memoria compacta.

## Proposito

Esta politica define como usar investigacion asistida sin contaminar las fuentes canonicas. NotebookLM puede ayudar a leer documentos largos, comparar fuentes y extraer ideas, pero sus respuestas no son autoridad del proyecto.

## Uso Correcto

- Analizar papers, docs externas o material largo.
- Obtener resumen inicial de una fuente externa.
- Preparar preguntas para una decision tecnica.
- Comparar alternativas antes de documentar una decision.

## Como Integrarlo

Cuando un hallazgo importa, no se copia completo sin revision. Se resume, se valida y se registra en el documento correcto. Si afecta comportamiento, abrir o actualizar un cambio OpenSpec. Si afecta arquitectura, documentarlo en `docs/architecture/system.md` o un ADR.

## Regla Breve

NotebookLM informa; no decide.
